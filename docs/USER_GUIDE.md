# TiebaMecha 用户指南

## 拟人化养号引擎 (BioWarming™) 详解

养号引擎是 TiebaMecha 的核心组件之一，通过模拟真人用户在百度贴吧的行为画像，大幅提升账号权重并平衡账号的“发帖/看帖”比重，从而降低被系统拦截或封禁的风险。

### 1. 工作原理
引擎通过后台调度（默认每 4 小时运行一次），模拟一个真实用户日常逛贴吧的过程。包括：进入贴吧、滚动翻页、随机点击帖子、阅读内容（停留）、随机点赞等全套动作。

### 2.针对新号/白号的加固支持 (新功能) [ENHANCED]
对于刚导入的、没有任何关注贴吧的新账号，引擎将自动启用以下特殊机制：

*   **冷启动探索模式 (Cold Start)**：
    *   **公域发现池**：当检测到账号关注列表为空时，系统不再跳过维护，而是自动从预设的“安全公域池”（包含王者荣耀、显卡、数码、Steam、电影等大流量板块）中随机挑选主题进行探索。
    *   **画像构建**：通过对公域池的主动阅读，账号将逐步并在大数据中形成活跃画像。

*   **动态行为自适应 (Dynamic Weighting)**：
    *   **状态判定**：账号关注贴吧数量 <= 3 时，自动判定为“发育期（Cold State）”。
    *   **潜行阅读机制**：发育期账号将**显著延长驻留耗时**（增加 1.5 - 2 倍），以通过拉长行为周期来规避高频检测。
    *   **低风险交互**：发育期账号的随合互动（点赞）概率被削减至极低（约 5%），确保在账号尚未成熟前不触发任何互动类风控。

*   **破冰式自动关注 (Ice-breaking)**：
    *   当新号在公域探索的过程中完成深度阅读后，系统有一定概率（约 25%）主动执行「关注」该公域贴吧的动作。
    *   通过此机制，新号在挂机运行数日后，将自动沉淀出数个大众化的关注贴吧，使其在账号中心看起来更加真实，成功渡过风控敏感期。

### 3. 使用建议
*   **保持服务在线**：拟人化养号是在后台守护进程中自动运行的，请确保程序或容器处于持续运行状态。
*   **新号耐心观察**：新号在最初的 1-3 天内可能不会有明显的权重提升，建议不要一开始就用新号进行高频发帖测试。
*   **查看日志**：在 Web 端日志中，您可以实时观察到 `[BioWarming]` 运行的状态，包括进入的贴吧名称以及是否触发了破冰关注。
